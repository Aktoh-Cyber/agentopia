# Agent Generator

Python-powered agent generator that creates and deploys new agents via GitHub integration

🔗 **Live Demo**: https://generator.mindhive.tech

## Features

- 🏭 **Agent Generation**: Creates new Python and JavaScript agents from requirements
- 🔗 **GitHub Integration**: Automatically commits generated agents to repository
- 🤖 **Router & Specialist Agents**: Supports both agent types
- 🚀 **Automated Deployment**: Generated agents trigger CI/CD for deployment
- 🐍 **Python-powered**: Built on Cloudflare Workers with Python
- 💾 **Response caching**: Improved performance with 30-minute cache
- 🎨 **Conversational Interface**: Natural language agent creation
- 📡 **RESTful API**: Programmatic agent generation endpoint

## Deployment

1. Set your Cloudflare API token:
   ```bash
   export CLOUDFLARE_API_TOKEN="your-token-here"
   ```

2. Set GitHub token for agent generation:
   ```bash
   cd agents/agent-generator
   wrangler secret put GITHUB_TOKEN
   ```

3. Deploy:
   ```bash
   ./scripts/deploy.sh
   ```

## GitHub Setup

The agent generator requires a GitHub Personal Access Token with the following permissions:
- **Repository permissions**: Contents (Read/Write), Pull requests (Write), Metadata (Read)
- **Account permissions**: None

Create a token at: https://github.com/settings/personal-access-tokens/new
