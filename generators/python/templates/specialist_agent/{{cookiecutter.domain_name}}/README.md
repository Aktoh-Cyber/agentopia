## Deployment

### Using Wrangler (Recommended)

1. Install Wrangler:
   ```bash
   npm install -g wrangler
   ```

2. Set up your Cloudflare API token:
   ```bash
   export CLOUDFLARE_API_TOKEN="your-token-here"
   ```

   ```bash

   ```

If you prefer to deploy manually:


### Key Features

1. **Native Python**: Uses Python standard library for maximum compatibility (third-party packages supported via Pywrangler)
2. **FFI Integration**: Seamless access to JavaScript APIs via Pyodide FFI
3. **Type Safety**: Full typing support with mypy compatibility
4. **Async Support**: Built on async/await for optimal performance
5. **Error Handling**: Robust error handling with logging

### Performance Optimizations

- **Cold Start Optimization**: Pyodide snapshot at deploy time

## Troubleshooting

### Common Issues

**Import Errors**
- Ensure `agent_framework` directory is present
- Check Python syntax compatibility
- Verify FFI imports are correct

**Deployment Failures**
- Confirm `python_workers` compatibility flag is set
- Check account and zone IDs in wrangler.toml
- Verify API token permissions

**Performance Issues**
- Monitor cold start times in Wrangler logs
- Check cache hit rates
- Optimize system prompt length
